Linux Administration Lab
Project Overview

The Linux Administration Lab is a hands-on project that demonstrates fundamental Linux system administration skills using command-line tools. The project simulates real-world server management tasks such as file and directory organization, user account creation, permission and ownership control, package installation, log monitoring, and basic automation with Bash scripting. It reflects typical responsibilities of junior system administrators and DevOps practitioners.

Tools Used

Ubuntu Linux

Bash Shell

Nginx / Git

Linux Terminal Commands

Tasks Performed

Created directories and files for structured storage

Added multiple users and assigned passwords

Configured permissions using chmod

Changed file ownership using chown

Installed packages (Nginx / Git)

Viewed logs in /var/log

Wrote a basic Bash script for system checks
